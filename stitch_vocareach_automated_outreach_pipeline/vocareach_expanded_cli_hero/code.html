Fault-Tolerant Execution Engine

Built to survive connection instability and strict rate-limit policies.

Exponential Backoff

Automatic retries with randomized jitter ensure transient API failures are handled gracefully.

Rate Limiting

Token bucket implementation throttles requests based on upstream provider X-RateLimit headers.

State Serialization

Pipeline state is serialized after each stage. Crash recovery resumes execution from last checkpoint.
